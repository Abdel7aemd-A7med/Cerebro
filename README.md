# 🧠 Cerebro : AI-Driven Database Architect

**Cerebro** is a Agent System built with **LangGraph** and **Ollama**. It is specifically designed to architect, manage, and query SQLite databases using natural language. The system doesn't just write code; it plans, secures, executes, and self-corrects in a seamless autonomous loop.

## 🚀 Key Features
- **Smart Task Decomposition:** Automatically splits complex requests (e.g., "Build a full banking system") into logical execution phases.
- **Autonomous Self-Correction:** Detects SQL execution errors and sends them back to the Architect node for immediate fixing.
- **Security First:** Includes a dedicated Guard Node to sanitize inputs and protect your local data from malicious injections.
